エンディング到達時のアチーブメント表示
===

## タスク概要
- `scene 14` または `scene 15` 到達時に、アチーブメント表示 UI を出す。
- 表示対象は `トゥルーエンド` `バッドエンド` `全アイテムコンプ` の 3 つ。
- 取得済み / 未取得を同時表示して見分けられる状態にする。

## 今回の要望 (統合者まとめ)
- エンディング到達時に、味気ない単文ではなく UI としてアチーブメントを表示したい。
- 取得できたものとできなかったものを、同じ画面内で判別可能にしたい。
