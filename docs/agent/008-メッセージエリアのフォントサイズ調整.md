# メッセージエリアのフォントサイズ調整

## 今回の要望
左側の列にある画像の上に重なっているメッセージエリアのフォントサイズが大きすぎるので、右側の列にあるラベル（"インベントリ - アイテムを使用できます"）と同じくらいにしてほしい。

## 詳細設計

### 変更箇所
対象ファイル: `webapp/frontend-react/src/pages/HomePage.tsx`

### 変更内容
左側の画像のオーバーレイにあるメッセージ（"さてさて..." の部分）の `Typography` コンポーネントの `variant` を、右側の "インベントリ..." と同じ `subtitle1` に変更するよ！

```tsx
// 変更前
<Typography variant="h6" sx={{ mb: 1.5 }}>
  {viewModel?.scene.text || dummyText}
</Typography>

// 変更後
<Typography variant="subtitle1" sx={{ mb: 1.5 }}>
  {viewModel?.scene.text || dummyText}
</Typography>
```

これで左右のバランスが良くなるはず！
