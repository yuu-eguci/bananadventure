ウィンドウタブタイトルとアイコン設定
===

## タスク概要
- ブラウザタブのタイトルをデフォルトから `Bananadventure` へ変更する。
- ブラウザタブのアイコンを `webapp/frontend-react/src/assets/ui-image/banana-meter-icon.webp` に変更する。

## 今回の要望 (統合者まとめ)
- タブタイトルを `Bananadventure` にしたい。
- タブアイコンへ `banana-meter-icon.webp` を使いたい。
- 指定アイコンが使えない場合は、その理由を明示したい。

## 変更対象
- `docs/agent/013-ウィンドウタブタイトルとアイコン設定.md`
- `webapp/frontend-react/index.html`
- `webapp/frontend-react/src/main.tsx` (必要に応じて)

## 詳細設計
### 事前確認
- 指定ファイル `webapp/frontend-react/src/assets/ui-image/banana-meter-icon.webp` は存在するため利用可能。
- 互換 fallback として `webapp/frontend-react/src/assets/ui-image/banana-meter-icon.png` も存在する。

### 実装方針
- タイトルは `webapp/frontend-react/index.html` の `<title>` を `Bananadventure` へ変更する。
- タブアイコンは `src/assets` から import して `main.tsx` で `link[rel~='icon']` を設定する。
- 主アイコンは `.webp` を使用し、必要時の fallback として `.png` も同時に登録する。
- 既存の `/vite.svg` 参照は削除し、ビルド時にハッシュ付き URL が反映される運用へ寄せる。
- `main.tsx` では既存 `link[rel='icon']` を再利用または置換し、重複追加を防ぐ。
- 適用順は `webp -> png fallback` とし、先に webp を登録する。

### 期待効果
- タブタイトルが全環境で `Bananadventure` 表示になる。
- 画像差し替え時にアイコン URL が更新され、ブラウザキャッシュの取り残しを減らせる。

### 検証観点
- ブラウザタブのタイトルが `Bananadventure` になっていること。
- ブラウザタブのアイコンが `banana-meter-icon.webp` へ差し替わること。
- `.webp` 非対応環境で `.png` fallback が有効になること。
