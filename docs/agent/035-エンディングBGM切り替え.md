# 035 - エンディング BGM 切り替え

## 要望

- 曲ファイルと曲名を定数で管理する
    - `banana-theme-64.mp3`: ばななちゃんのテーマ
    - `banana-theme-piano-64.mp3`: ばななちゃんのテーマ - ピアノ
- メイン画像エリアの外側の右上に、曲名を表示する
    - フォーマット: `♪ ばななちゃんのテーマ`
- エンディングのシーン (通常エンディングとゲームオーバーエンディング両方) では「ばななちゃんのテーマ」を再生
- それ以外のシーンでは「ばななちゃんのテーマ - ピアノ」を再生

## 現状

- メイン画像エリアの (重ねて) 右上に BGM OFF/ON ボタンがある
- `webapp/frontend-react/public/bgm/banana-theme-64.mp3` を再生している
