画像実装ガイド
===

## 背景
別のメンバーがこのプロジェクトへ画像を追加したいとき、どこに何を置けばいいかわからない。
画像の種類ごとに「配置場所・表示場所・推奨サイズ・アスペクト比」をまとめたドキュメントが必要。

## 今回の要望
- バナナメーターの画像
- インベントリのボタンに出る画像
- 左側の列のメイン画像

それぞれについて、どこに配置すれば・どこに出る・画像比率はこう、というのをドキュメント化する。

## ゴール
- 画像を追加したいメンバーが、このドキュメントだけ読めば迷わず作業できる状態にする。
- 各画像種別ごとに「ファイル配置場所」「JSON への書き方」「表示サイズ・比率」を明記する。

## 非ゴール
- 画像ファイル自体の作成・追加。
- 画像以外の UI 変更。

## 変更対象
- `docs/agent/009-画像実装ガイド.md` (このファイル)

## 詳細設計

画像を追加する際は、以下のガイドラインに従って配置と設定を行ってほしい。

### 1. メイン画像 (左側の大きな画像)

物語のシーンを彩るメインビジュアルだよ。

| 項目 | 内容 |
|------|------|
| ファイル配置場所 | `webapp/frontend-react/public/scene-image/` |
| JSON フィールド | `bananadventure-scenes.json` の各シーンオブジェクトの `image` フィールド |
| JSON の書き方 | `/scene-image/ファイル名.拡張子` (例: `/scene-image/scene-01.png`) |
| 表示サイズ | 幅 100%、高さ 520px (スマホ) 〜 560px (PC) |
| 推奨アスペクト比 | **16:9** 推奨 (`object-fit: cover` で枠いっぱいに表示されるため、横長が自然に見える) |
| 表示場所 | ホーム画面の左カラム全体 (md:8) |

**JSON の例:**
```json
{
  "id": 1,
  "image": "/scene-image/morning-room.png",
  "text": "ふああぁ、良く寝たのだ。",
  "sceneChoices": [...]
}
```

---

### 2. インベントリ画像 (アイテムのボタン画像)

手に入れたアイテムを表示する小さな画像だよ。

| 項目 | 内容 |
|------|------|
| ファイル配置場所 | `webapp/frontend-react/public/item-image/` |
| JSON フィールド | `sceneChoices[].itemsOnSelect[].image` フィールド |
| JSON の書き方 | `/item-image/ファイル名.拡張子` (例: `/item-image/banana-milk.png`) |
| 表示サイズ | 48px × 48px (スマホ) 〜 64px × 64px (PC) |
| 推奨アスペクト比 | **1:1 (正方形)** |
| 表示場所 | ホーム画面右カラムの「インベントリ」セクション内の各アイテムカード |
| フォールバック | 画像が読み込めないとき `/sample-image/sample.png` が表示される |

**JSON の例:**
```json
{
  "id": 2,
  "order": 2,
  "image": "",
  "text": "コンビニに駆け込む",
  "responseText": "豆乳バナナを買えた！",
  "bananaMeterDelta": 0,
  "nextSceneId": 2,
  "itemsOnSelect": [
    {
      "id": 0,
      "text": "豆乳バナナ",
      "image": "/item-image/banana-milk.png",
      "bananaMeterDelta": 20,
      "bananaMeterDeltaPercent": 0,
      "used": false
    }
  ]
}
```

---

### 3. バナナメーター画像 (UI アイコン)

バナナメーターの横に表示されるアイコンだよ。

| 項目 | 内容 |
|------|------|
| ファイル配置場所 | `webapp/frontend-react/public/ui-image/` |
| 設定方法 | **JSON ではなくコードで管理** (現状は `dummyImage` = `/sample-image/sample.png` が固定で使われている) |
| 変更方法 | `webapp/frontend-react/src/pages/HomePage.tsx` の `import dummyImage from "/sample-image/sample.png";` を変更するか、新しいパスを直接 `src` に指定する |
| 表示サイズ | 64px × 64px |
| 推奨アスペクト比 | **1:1 (正方形)** |
| 表示場所 | ホーム画面右カラムの「バナナメーター」カード内 (数値の上) |

**コードの該当箇所 (`HomePage.tsx`):**
```tsx
// 現状: dummyImage (= /sample-image/sample.png) が使われている
import dummyImage from "/sample-image/sample.png";

// バナナメーター表示部分
<img
  src={dummyImage}
  style={{ width: 64, height: 64, marginBottom: 8 }}
/>
```

画像を変えたいときは、`dummyImage` の import 先を変えるか、`src` に直接パスを書けばいいよ。

---

### ディレクトリ構成まとめ

```
webapp/frontend-react/public/
├── scene-image/        ← メイン画像 (シーンごとの背景)
│   └── scene-01.png    (例)
├── item-image/         ← インベントリ画像 (アイテムのアイコン)
│   └── banana-milk.png (例)
├── ui-image/           ← UI 画像 (バナナメーターアイコンなど)
│   └── banana-icon.png (例)
├── jingle-gif/         ← ローディング GIF (既存)
│   └── loading-00001.gif
└── sample-image/       ← サンプル・フォールバック画像 (既存)
    └── sample.png
```
