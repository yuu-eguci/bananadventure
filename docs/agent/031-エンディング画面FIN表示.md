# 031 - エンディング画面 FIN 表示

## 今回の要望

エンディング画面に "FIN" 的な表示がほしい。終わったのか分かりづらいので。

エンディングのシーンでは、わかりやすくしたいね。シーンメッセージの右下とかに "FIN" だか "おしまい" だか "アチーブメントを確認してね" だか、出せばいいかな。

## 背景・現状

- `HomePage.tsx` には `isEndingScene` という boolean がすでにある
  - `currentSceneId === ENDING_SCENE_IDS.TRUE || currentSceneId === ENDING_SCENE_IDS.BAD`
  - `ENDING_SCENE_IDS.TRUE = 14`, `ENDING_SCENE_IDS.BAD = 15`
- エンディングに到達すると右カラムにアチーブメントカードが表示される
- シーンメッセージエリア (左カラム) には特に "終わった" を示す要素がない
