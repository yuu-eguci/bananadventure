# 036 - エンディング BGM 割り当て修正

## 要望

- 035 で実装した BGM 切り替えの曲の割り当てが逆だったので修正する
- 正しい割り当て:
    - エンディングのシーン (通常エンディングとゲームオーバーエンディング両方) -> `banana-theme-piano-64.mp3` (ばななちゃんのテーマ - ピアノ)
    - それ以外 -> `banana-theme-64.mp3` (ばななちゃんのテーマ)

## 現状

- 035 で実装済みだが、 `BGM_TRACKS` の `main` と `ending` の `src` と `label` が逆になっている

## 詳細設計

### 修正対象

- `webapp/frontend-react/src/hooks/useBgmPlayer.ts` の `BGM_TRACKS` 定数

### 現状 (間違ってる)

```ts
export const BGM_TRACKS = {
  main: {
    src: "/bgm/banana-theme-piano-64.mp3",
    label: "ばななちゃんのテーマ - ピアノ",
  },
  ending: {
    src: "/bgm/banana-theme-64.mp3",
    label: "ばななちゃんのテーマ",
  },
} as const;
```

`main` にピアノ版、 `ending` に通常版が入っちゃってて、あべこべになってるね。

### 修正後 (正しい割り当て)

```ts
export const BGM_TRACKS = {
  main: {
    src: "/bgm/banana-theme-64.mp3",
    label: "ばななちゃんのテーマ",
  },
  ending: {
    src: "/bgm/banana-theme-piano-64.mp3",
    label: "ばななちゃんのテーマ - ピアノ",
  },
} as const;
```

- `main` (通常シーン) -> `banana-theme-64.mp3` (ばななちゃんのテーマ)
- `ending` (エンディングシーン) -> `banana-theme-piano-64.mp3` (ばななちゃんのテーマ - ピアノ)

要は `src` と `label` のペアを `main` と `ending` で入れ替えるだけ！他のロジックには触らないよ。
