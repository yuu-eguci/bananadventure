# 設定資料集更新と音声よここ追加

## タスク概要
- 目的: `/lore` の設定資料集を、内容更新・キャラクター表示・音声再生機能の3点で拡張する
- 背景: 既存の `037-設定資料集ページ追加` は単一QAのみで、今回の要望（Q/A拡張・よここ表示・音声ON/OFF）を満たしていない

## 今回の要望（原文要約）
- 設計ノート 037 相当: 内容追加

  - Q3の回答末尾に YouTube 風アイコン付きリンクを表示し、`https://www.youtube.com/@うみにょ·uminyo` へ遷移させる
- 設計ノート 038 相当: キャラクター「よここ」追加
  - A表示エリアで、回答テキスト左側に `001-yokoko.webp` を表示する
  - 回答をキャラクターが話している見せ方にする
- 設計ノート 039 相当: 音声追加
  - `/lore` に `音声 ON/OFF` 機能を導入する（`/` の BGM ON/OFF と同様の操作感）

  - 再生順は以下:
    - Q1: `001_Q1.mp3` -> `002_A1.mp3`
    - Q2: `003_Q2.mp3` -> `004_A2.mp3`
